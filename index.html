Setup steps
1
Deploy the server
Go to render.com → New Web Service → connect your GitHub repo with the server.js file → deploy free
2
Add your Anthropic API key
In Render dashboard → Environment → add variable:
ANTHROPIC_API_KEY = your_key_here
3
Set TradingView webhook URL
When creating your alert → Notifications tab → Webhook URL:
https://your-app.onrender.com/webhook
4
Enable push notifications
Tap below to allow notifications so your phone buzzes when a signal fires
5
Add to home screen
Chrome menu → Add to Home Screen → "Lucid Edge" → one tap to open anytime
